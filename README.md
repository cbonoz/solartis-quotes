Solaris Insurance API
---

Tired of online forms?
Get Insurance Quotes from the command line!

### Getting Started:
* Download the repo
* Set your api token locally:
<pre>
    SOLARIS_TOKEN=<token>
</pre>
* Install dependencies:
<pre>
    npm install
</pre>
* Start the app
<pre>
    npm start
</pre>
